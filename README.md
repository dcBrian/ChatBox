## Getting Started

### Prerequisites

-   yarn
    ```sh
    npm install --global yarn
    ```

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/ivucady/ChatBox.git
    ```
2. Install all depedencies
    ```sh
    yarn install
    ```
3. Launch
    ```JS
    yarn start
    ```
